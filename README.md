## Backend Setup
#### Open a terminal and navigate to the Backend folder:
cd Backend

### Install dependencies:
npm install

### Start the backend server:
npm run dev


## Frontend Setup
### Open a new terminal and navigate to the forntend folder:
cd forntend

### Install dependencies:
npm install

### Start the frontend development server:
npm run dev
